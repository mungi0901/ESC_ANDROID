# 📱 Android App Development Study

## 🧑‍💻 스터디 소개

본 스터디는 **Android Studio와 Jetpack Compose**를 활용하여
앱 개발의 기초부터 협업 프로젝트까지 경험하는 것을 목표로 합니다.

단순히 앱을 만드는 것을 넘어,
**AI가 생성한 코드를 이해하고 수정할 수 있는 개발자**가 되는 것을 목표로 합니다.

---

## 🎯 스터디 목표

* Android 앱 구조 이해 (Activity, Lifecycle, Compose)
* Kotlin 기반 UI 개발 경험
* Git/GitHub 협업 경험
* API 통신 및 데이터 처리 이해
* 팀 프로젝트를 통한 실전 개발 경험
